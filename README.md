# MERN Blog App

## 🚀 Live Demo

Frontend: https://mern-blog-red-mu.vercel.app/
Backend: https://mern-blog-inqk.onrender.com

## 📌 Features

* User authentication (JWT)
* Create, edit, delete posts
* Comment system
* Protected routes

## 🛠️ Tech Stack

* MongoDB
* Express
* React (Vite)
* Node.js

## ⚙️ Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
